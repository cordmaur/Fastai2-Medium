Método de Análise (1) Qualitativo
(1) O método quantitativo considera os danos e os tempos de retorno associado aos desastres para estimar o risco. O método qualitativo é utilizado quando não há estação hidrológica representativa para o município. Nesses casos, o risco é atribuído por uma matriz de riscos, conforme o número de eventos e respectivos danos reportados.
